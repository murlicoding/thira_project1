📊 Data Cleaning & Visualization Project
📌 Overview

This project demonstrates the complete process of cleaning, preprocessing, analyzing, and visualizing a raw dataset using Python. It focuses on transforming messy data into a clean and structured format while extracting meaningful insights through Exploratory Data Analysis (EDA) and visualizations.

The project showcases essential data analytics skills, including handling missing values, removing duplicates, detecting outliers, and creating informative charts to support data-driven decision-making.

🚀 Features
Load and analyze raw CSV datasets
Handle missing values using mean/mode imputation
Remove duplicate records
Detect and remove outliers using the IQR method
Perform Exploratory Data Analysis (EDA)
Generate summary statistics
Create multiple visualizations:
📈 Histogram
📦 Box Plot
📊 Bar Chart
🔥 Correlation Heatmap
🎯 Scatter Plot
Export the cleaned dataset for future analysis

⚙️ Installation

Clone the repository:

git clone https://github.com/murli_rathi_0/Data-Cleaning-Visualization.git

Navigate to the project directory:

cd Data-Cleaning-Visualization

Install the required libraries:

pip install pandas numpy matplotlib seaborn
▶️ Usage
Place your dataset (dataset.csv) in the project folder.
Run the Python script:
python data_cleaning.py
The program will:
Clean the dataset
Handle missing values and duplicates
Remove outliers
Generate visualizations
Save the cleaned dataset as cleaned_data.csv
